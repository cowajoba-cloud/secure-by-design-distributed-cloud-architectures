## 3. Methodological Patterns Observed

Across the reviewed papers, the following evaluation trends were observed:

- Most studies rely on simulated datasets rather than live distributed environments.
- Many focus on architecture design rather than empirical detection benchmarking.
- Evaluation metrics commonly include detection rate and false positives.
- Few studies measure detection latency under distributed node conditions.
- Limited discussion of log pipeline delay in multi-node systems.

## 4. Recurring Limitations in Literature

The following recurring weaknesses were identified:

1. Lack of empirical validation in small-scale distributed lab environments.
2. Minimal quantitative benchmarking of detection latency.
3. Insufficient correlation accuracy measurement under network delay.
4. Over-reliance on theoretical security models.
5. Limited comparison between rule-based and behavioral detection systems.

## 5. Emerging Research Gaps

Based on preliminary review, potential research gaps include:

- Absence of structured detection latency benchmarking frameworks in distributed cloud environments.
- Limited analysis of SIEM performance under adversarial stress conditions.
- Lack of standardized methodology for validating detection pipelines in hybrid infrastructures.
- Minimal empirical evaluation of open-source detection engines in distributed architectures.

This suggests a gap in measurable detection validation methodologies within practical cloud-native deployments.

---

## 6. Preliminary Problem Framing

Current literature provides extensive discussion on secure architecture design but insufficient empirical validation of detection effectiveness in distributed environments.

There is limited quantitative research on:

- Detection latency behavior across distributed nodes.
- Correlation rule performance under adversarial load.
- Log ingestion integrity under network segmentation.
- Comparative evaluation of detection mechanisms in practical lab conditions.

This research aims to contribute by:

Developing and experimentally validating a structured detection benchmarking framework within distributed cloud architectures.
